Daily Insight

AI-driven Daily Market Summary & Stock Analysis

Team 12 — SWPP Project 2025

Introduction & Features

Motivation

개인 투자자는 매일 넘쳐나는 뉴스, 재무 데이터, 시장 지표를 직접 분석하기 어렵다. Daily Insight는 자동 크롤링 + LLM 분석을 결합하여 누구나 매일 아침 시장의 핵심만 빠르게 이해하도록 돕는다.

Key Features

  • 전 종목 요약 제공 (기본적/기술적/뉴스 기반)
  • 산업/규모 필터링 기능
  • 시장 전체 분석(General, KOSPI, KOSDAQ)
  • Watchlist 기반 개인화 기능
  • Guest 모드 지원

Competitive Landscape

여러 서비스가 데이터 일부만 제공하는 반면, Daily Insight는 시장·뉴스·재무·분석을 통합적으로 제공한다.

User Interface

앱은 Home(요약 목록), Index(KOSPI/KOSDAQ), Detail View, My Profile로 구성된다.

System Architecture

Overview

Crawler → S3 저장 → LLM 분석 → Backend API → Android Client

LLM Structure

GPT-5 기반 HyDE RAG를 적용하여 고정밀 종목 분석을 생성한다.

Challenges & Solutions

데이터 품질 문제

뉴스, 재무 데이터의 불규칙성 해결 → 전처리 + 중복 제거 + 안정화 파이프라인 구축

LLM 분석 안정성

HyDE RAG와 구조화된 Input Schema로 편차 없이 일관된 분석 생성

성능 및 응답 속도

SharedMemory + S3 캐시로 요청당 1초 이내 응답을 보장

Testing

Backend Coverage

백엔드 전체 커버리지 99% 달성

Android Testing

DTO/Repository는 높은 커버리지 확보, UI/Room은 Instrumented Test 필요

Project Management

Processes

  • Iteration 관리 (Kickoff → Mid → Wrap-up, 총 5 Iteration)
  • Daily Scrum, Weekly Milestone 점검
  • CI 없이 수동 배포 (AWS EC2)
